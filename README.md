# Calculator App

A simple calculator web application that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features
- Basic arithmetic operations (+, −, ×, ÷)
- Clear (AC) and delete (DE) functionality
- Clean and simple UI

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM manipulation)
  
## Live Demo
Check out the live version of the calculator here: [Live Demo](https://sharaf-ahmed-dev.github.io/calculator-app/)

## ⚠️ Note
- This project uses JavaScript `eval()` for calculations.
